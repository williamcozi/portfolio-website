프로젝트  03  ·  식품 콜센터 발화문 분석 프로젝트
LDA vs BERTopic — 분석 결과 및 챗봇 구축
NLP 프로젝트
📊  LDA (Latent Dirichlet Allocation)
• 전통적 확률 기반 토픽 모델링 기법
• 단어 빈도 기반으로 잠재 주제를 추출
• 문맥 파악에 한계 — 동음이의어 구분 어려움
• 기준선(Baseline) 모델로 활용
🚀  BERTopic ( 논문 기반 적용)
• BERT 임베딩 + c-TF-IDF 결합 기법
• 문맥적 의미를 LDA보다 정교하게 포착
• 관련 논문 직접 탐독하며 모델 이해 심화
VS
🎯  토픽 모델링 주요 결과
• 고객 발화의 주요 주제와 패턴 정교하게 도출
• 반복적 질문 · 주요 불만 사항 유형화
• 배송·포장·결제 관련 문의가 전체의 60%↑ 차지
• BERTopic이 식품 도메인 문맥 파악에서 우위 확인
🤖  FAQ 챗봇 시스템 구축
• 분석 결과 기반 FAQ 항목 정리 및 구조화
• 자동 Q&A 챗봇 설계로 즉각적인 고객 응대
• 상담 효율 향상 시스템 구축 · 실용적 결과 도출
전처리 → 모델 학습 → 결과 분석 → 실서비스 적용까지 NLP 전 과정 경험
